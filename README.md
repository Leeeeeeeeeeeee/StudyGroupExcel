# Daily

- Input
- 录入数据
- 排序
- 检索没有出现的人名，选择是缺勤（-1）还是请假（0），需要数据库中一个memberList的表
- 把数据放进数据库中每个成员的表，判断是否需要退出
- 删除退出成员的表
- 把数据放进数据库中每日的表
- 输出每日的.md文件 格式：日期+排序表+请假人员+缺勤人员+需要退出的人员

# Weekly

- 录入数据库中各个成员前7天的表

- 总计各个成员前七天的番茄总数

- 以前七天的番茄总数做一个排序

- 创建一个表，降序录入id+成员名+七日各自的番茄数+番茄总数

- 输出每周的.csv文件
